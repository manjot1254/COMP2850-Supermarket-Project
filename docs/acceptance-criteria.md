# Acceptance Criteria

## Product Browsing

Given products exist in the system  
When a customer opens the homepage  
Then available products are displayed  


## Add to Basket

Given a product is available  
When the customer selects "Add to Basket"  
Then the item appears in the basket  


## Checkout

Given a customer has items in their basket  
When checkout is completed  
Then an order confirmation is generated  


## Order History

Given previous orders exist  
When a customer accesses their account  
Then past orders are displayed  


## Picking List

Given an order exists  
When warehouse staff access the dashboard  
Then picking lists are displayed  


## Stock Update

Given stock quantities change  
When warehouse staff update inventory  
Then database stock levels are updated  


## Analytics Dashboard

Given sales data exists  
When a manager opens the dashboard  
Then analytics information is displayed
