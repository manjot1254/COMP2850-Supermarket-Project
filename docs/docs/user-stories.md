# User Stories

## Customer User Stories

As a customer, I want to browse products so that I can choose items to purchase.

As a customer, I want to search for products so that I can quickly find items.

As a customer, I want to view product details so that I can make informed decisions.

As a customer, I want to add items to a basket so that I can purchase multiple products.

As a customer, I want to remove items from my basket so that I can modify my order.

As a customer, I want to checkout securely so that I can complete my purchase.

As a customer, I want to receive order confirmation so that I know my purchase was successful.

As a customer, I want to view my order history so that I can track past purchases.


---

## Warehouse Operative User Stories

As a warehouse operative, I want to view picking lists so that I can prepare customer orders efficiently.

As a warehouse operative, I want to update stock levels so that inventory remains accurate.

As a warehouse operative, I want to mark orders as packed so that delivery can proceed.

As a warehouse operative, I want to report damaged or missing items so that issues can be resolved quickly.

As a warehouse operative, I want to view substitution suggestions so that unavailable items can be replaced.

As a warehouse operative, I want to manage stock additions and removals so that warehouse data stays updated.


---

## Manager / Marketing User Stories

As a manager, I want to view sales analytics so that I can monitor business performance.

As a manager, I want to monitor warehouse activity so that operations run efficiently.

As a manager, I want to view stock levels so that shortages are avoided.

As a manager, I want to verify deliveries so that customer orders are fulfilled correctly.

As a manager, I want to view performance reports so that business decisions can be improved.
