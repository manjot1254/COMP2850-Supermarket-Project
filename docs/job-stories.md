# Job Stories

When browsing products,
I want clear product categories,
So I can quickly find items I need.

When searching for products,
I want accurate search results,
So I can locate items efficiently.

When adding items to my basket,
I want instant feedback,
So I know the action was successful.

When completing checkout,
I want confirmation information,
So I know my order has been placed.

When preparing warehouse orders,
I want organised picking lists,
So I can work efficiently.

When stock becomes low,
I want notifications,
So restocking can occur quickly.

When items are unavailable,
I want substitution suggestions,
So customer orders can still be fulfilled.

When reviewing business performance,
I want analytics visualisations,
So I can make informed decisions.
